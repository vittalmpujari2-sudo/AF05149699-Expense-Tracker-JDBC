# Expense Tracker System (Java JDBC)

This is a console-based Expense Tracker application that helps track and analyze expenses using Java and JDBC.

## 🚀 Features
- Add and delete expenses
- View all expenses
- Category-wise spending analysis
- Monthly expense summary
- Budget setting and tracking

## 🛠 Technologies Used
- Java
- JDBC
- MySQL

## 🧠 Concepts Used
- SQL aggregation (SUM, GROUP BY)
- Budget comparison logic
- Data analysis using database queries

## ▶️ How to Run
1. Create database in MySQL
2. Update DB credentials
3. Run Main.java

## 📌 Highlights
- Expense analysis system
- Budget tracking feature
- Simple and efficient design
